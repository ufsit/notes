# cron
* `crontab -e` for editing the current user's crontab
* Location of all user crontabs
  * `/var/spool/cron/crontabs`
* Location of system-wide crontabs
  * `/etc/cron*`
  * `/etc/cron.d`
  * `/etc/cron.daily`
  * `/etc/cron.hourly`
  * `/etc/cron.weekly`
  * `/etc/cron.monthly`
  * `/etc/crontab`
    * it can specify what user can run a cronjob

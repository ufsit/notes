# Command cheat sheet
```grep pattern file``` = ```Select-String -Pattern patter file``` search in files  
```find /path -name "*.txt"``` = ```Get-Children -Path /path -Recurse -Filter *.txt -ErrorAction SilentlyContinue``` find files (ex .txt files)  
```ps``` and ```top``` = Get-Process  
```kill PID``` = ```Stop-Process -Id PID -Force```  
```df -h``` =  ```Get-PSDrive``` disk usage  
```free -h``` = ```Get-CimInstance Win32_OperatingSystem``` tot visable mem  
```uname -a``` = ```Get-ComputerInfo```  
```ip a``` = ```Get-NetIPAddress```  
```netstat -tuapn``` = ```Get-NetTCPConnection``` and ```Get-NetUDPEndpoint``` show active connections  
```curl url``` = ```Invoke-WebRequest url``` fetch web content   
```wget url``` = ```Invoke-WebRequest url -OutFile file``` download file  
```id``` = ```whoami /all``` show usr privs and groups  
```env``` = ```Get-ChildItem Env:``` list environment vars  
```which cmd``` = ```Get-Command cmd```  
```ip route``` = ```Get-NetRoute``` show routing table  
```dig domain``` and ```nslookup``` = ```Resolve-DnsName domain```  
```ps -aux``` = ```Get-Process```  
```service name status``` = ```Get-Service -Name name```  
```service name stop/start``` = ```Stop-Service -Name name```/```Start-Service -Name name```  
```who -u``` = ```query user```  
```adduser name``` = ```New-LocalUser name```  
```passwd user``` = ```Set-LocalUser -Name user -Password (Read-Host -AsSecureString)```  
```userdel name``` = ```Remove-LocalUser name```  
```journalctl -xe``` = ```Get-WinEvent -LogName System -MaxEvents 20``` view sys logs  
```tail -f /var/syslog``` = ```Get-Content -Path C:\Windows\System32\winevt\Logs\System.evtx -Wait``` moniter live logs  
```alias``` = ```Get-Alias```  ```Set-Alias alias cmd``` ```Remove-Item Alias:alias```
```grep``` = ```findstr```
